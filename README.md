# Decode Morse Code <a name="about-project"></a>

**Decode Morse Code** is a Ruby programme that decodes a coded message written in morse code.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Ruby

### Key Features <a name="key-features"></a>

- Decoding a letter
- Decoding a word
- Decoding a message
